# Examples

This directory contains example PDFs and their generated HTML outputs.

## Sample Files

- `sample.pdf` - A sample PDF document (to be added)
- `sample-output.html` - Example HTML output from processing sample.pdf

## Try It Yourself

1. Add your own PDF to this directory
2. Run the skill:
   ```bash
   python ../scripts/main.py your-document.pdf
   ```
3. Open the generated HTML file in your browser

## Example Outputs

The generated HTML files demonstrate:
- Interactive navigation
- Search functionality
- Dark/light mode
- Responsive design
- Clean formatting

Open any `.html` file in this directory with your browser to see the results.
